# LetsChat

**LetsChat** is a simple group chat application that allows users to communicate and interact with one another. Users can create posts and respond to messages sent by other users within a shared chat environment.

---

##  Features

- Group chat communication  
- Create and post messages  
- Reply to messages from other users  
- Simple and intuitive user interface
